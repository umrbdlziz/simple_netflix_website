# Movie-Search-Web

First ReactJs project that allow user to search for movie 

## Run Locally

Clone the project

```bash
  git clone https://github.com/umrbdlziz/simple_netflix_website.git
```

Go to the project directory


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
